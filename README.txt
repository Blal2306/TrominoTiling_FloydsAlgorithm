***** RUNNING INSTRUCTIONS *****

navigate to the directory containing all the source files, and type "make" to make three executable files

PROBLEM # 1 : Matrix Multiplication
	type "p1" followed by integer k, this will run the program using 2^k * 2^k matrix
	For example, "p1 2"

PROBLEM # 2 : Tromino tiling
	type "p2" followed by integer k, this will run the program using 2^k * 2^k board
	For example, "p2 2"

PROBLEM # 3 : Floyd's algorithm
	type "p3" followed by name of the input file that contains the adjacency matrix.
	For example, "p3 input.txt"
	The format of the input file contains comma seperated values for each row
